# al-input 

## Simple Text Input
```html
<al2-input
    type="text"
    label="Text Input">
</al2-input>
```

## With Hint Text
```html
<al2-input
    type="text"
    label="Text Input"
    hint="Give extra information here">
</al2-input>
```

## With Tooltip
```html
<al2-input
    type="text"
    label="Text Input"
    tip="Give extra information here">
</al2-input>
```

## Marking Required Fields

```html
<al2-input
    type="text"
    label="Text Input"
    required="true">
</al2-input>
```

## Disabled Fields

```html
<al2-input
    type="text"
    label="Text Input"
    disabled="true">
</al2-input>
```
