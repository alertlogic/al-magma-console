Search Controls

Component Name:
al-search-bar

Al Search bar with autofocus

Add [autofocus]="true" to the component to autofocus on that input field.

Header search with larger icon

Add [searchIcon]="'header'" to the component for a larger search icon.

Al Search bar with label

Add [showLabel]="true" to the component for the float label.

Al Search bar without label

Al Search bar with regex input validation

Add regexPattern="YOUR_REGEX_HERE" to the component for the regex validation to be enabled.

Add invalidPatternText="YOUR_MESSAGE_GOES_HERE" to the component to customise the error message.
